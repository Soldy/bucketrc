# bucketrc

 pullable bucket.

## init

```javascript

const bucket = new (require('bucketrc')).base();

```

## add

```javascript

bucket.add('element');
//true

```

## first

```javascript

bucket.first();
//first element

```

## last

```javascript

bucket.las();
//last element

```

## random

```javascript

bucket.random();
//random element

```

## all

```javascript

bucket.all();
//all elements

```

## remove

```javascript

bucket.remove(1);
//remove element 1

```
